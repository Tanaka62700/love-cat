# LoveCat Game Pitch

This project is a polished landing page for a cozy, social, and collectible game concept called LoveCat.

## What is included

- A modern hero section for introducing the game
- Feature cards for collecting, exploring, building, and playing with friends
- A visual gallery inspired by the game world
- A small interactive surprise box to make the page feel more alive

## How to view it

You can open [index.html](index.html) directly in your browser. The page now includes a real local trailer video at `public/intro.mp4`.

No Python or build tools are required. The project is built using plain HTML, CSS, and JavaScript.
